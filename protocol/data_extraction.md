# Data Extraction Template (Fields)

## Bibliographic
- Record ID
- Title
- Authors
- Year
- Venue (journal/conference/preprint)
- DOI / URL

## Data & Modalities
- Modalities (e.g., scRNA-seq, scATAC-seq, CITE-seq)
- Multi-modal alignment strategy
- Dataset(s) used for pretraining
- Dataset size (cells, donors, tissues)

## Model
- Model name
- Architecture (transformer, VAE, diffusion, etc.)
- Foundation-model evidence (pretraining scale/transfer)
- Objective (masked modeling, contrastive, generative)
- Tokenization / feature representation

## Evaluation
- Downstream tasks (annotation, batch correction, imputation, perturbation)
- Benchmarks and metrics
- Baselines compared

## Resources
- Code availability
- Data availability
- License

## Notes
- Key findings
- Limitations
